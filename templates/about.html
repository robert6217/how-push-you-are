WHAT

What is this system?
This is a website system. When you log in with Facebook, we will use the your Facebook profile picture to do the analysis. After the system analysis, you will know HOW PUSH YOU ARE in PTT.
  • 1. User browse HOWPUSH website, and login.
  • 2. HOWPUSH request the server.
  • 3. Call Facebook API to get user’s Token.
  • 4. Server get user’s Token.
  • 5. Server store the user information, include userID, name, token, picture(profile picture).
  • 6. Response the user information of the specified Token.
  • 7. Predict user’s profile picture will get how many 'push'.
  • 8. Return to HOWPUSH website
  • 9. User get the predict information about 'push'.

WHY

Why do this?
We are master student in Tamkang University, we have a 'Big Data Mining' course in the spring semester of 2018, this is our final project.
In this course, we learned a lot about machine learning, deep learning and many big data analysis applications, and how to use these module, so that we can easily to train the model and predict it, as the teacher often said: “Three lines of code.” Among this, we are more interested in the application of CNN, so we want to train a model to be close to our lives, and finally decided to predict how many pushes in the Beauty of PTT. We also hope that in addition to practice, we can make a system and truly use our model.
