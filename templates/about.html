WHAT

What is this system?
This is a website system. When you log in with Facebook, we will use the your Facebook profile picture to do the analysis. After the system analysis, you will know HOW PUSH YOU ARE in PTT.
  • 1. User browse HOWPUSH website, and login.
  • 2. HOWPUSH request the server.
  • 3. Call Facebook API to get user’s Token.
  • 4. Server get user’s Token.
  • 5. Server store the user information, include userID, name, token, picture(profile picture).
  • 6. Response the user information of the specified Token.
  • 7. Predict user’s profile picture will get how many 'push'.
  • 8. Return to HOWPUSH website
  • 9. User get the predict information about 'push'.
